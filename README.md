# EchoHub
🎧 EchoHub

EchoHub is a modern, interactive podcast platform that combines content exploration with AI-powered creation. It allows users to discover, play, and even generate podcast-style content in a clean, visually engaging interface.

⸻

🚀 Overview

EchoHub is designed to make learning and exploration more engaging through audio. Users can browse curated podcast episodes, interact with a seamless audio player, and generate new content using an integrated AI Studio.

The platform blends design, functionality, and creativity into a single cohesive experience.

⸻

🌐 Features

🎯 Podcast Browsing
	•	Explore podcasts across categories like:
	•	Technology
	•	Science
	•	Geography
	•	History
	•	Each episode is presented as a visual card with custom cover art

⸻

🔍 Search & Filtering
	•	Search episodes using keywords or tags
	•	Quickly find relevant content with an intuitive interface

⸻

🎵 Persistent Mini Player
	•	Audio continues playing while navigating the site
	•	Includes:
	•	Play / Pause
	•	Seek bar
	•	Playback controls
	•	Maintains state across pages

⸻

📄 Episode Details
	•	Each episode includes:
	•	Title
	•	Description
	•	Tags
	•	Transcript
	•	Media support (audio/video)

⸻

🤖 AI Studio
	•	Generate podcast scripts from a single idea
	•	Convert text → speech using built-in voice playback
	•	Turns ideas into complete audio experiences instantly

⸻

⭐ Favorites & History
	•	Save favorite episodes
	•	Automatically track recently played content

⸻

🌗 Light / Dark Mode
	•	Toggle between themes
	•	Designed for comfort and accessibility

⸻

📤 Upload System
	•	Add new episodes with:
	•	MP3 file
	•	Title
	•	Category
	•	Duration
	•	Tags
	•	Stored locally in the browser

⸻

🧭 Website Structure
	•	Home → Featured categories and podcasts
	•	Category Pages → Topic-based podcast lists
	•	Episode View → Detailed content + playback
	•	AI Studio → Script generation interface
	•	About Page → Team + “About Us” podcast
	•	Mini Player → Always visible across pages

⸻

🎨 Design & UI
	•	Soft pastel color palette
	•	Clean modern typography
	•	Rounded cards and smooth layout
	•	Fully responsive design
	•	Dark mode inspired by a studio environment

⸻

⚙️ Technical Stack
	•	Frontend: HTML5, CSS3, JavaScript
	•	Layout: Flexbox + responsive design
	•	Audio: HTML5 <audio> with JS controls
	•	Storage: LocalStorage (favorites, history)
	•	Data: Local JS file (episodes, metadata)
	•	AI Integration: Script generation + text-to-speech
	•	Hosting: GitHub Pages (static site)
